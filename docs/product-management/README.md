# Tabrix Product Management Docs

本目录用于集中存放 `Tabrix` 的产品管理与任务治理文档，避免产品文档散落在 `docs/` 根目录中。

## 文档清单

- `PRODUCT_TASK_SYSTEM_AND_EXECUTION_QUEUE_zh.md`
  - 主任务系统文档，定义任务池、状态、版本与回写规则。
- `PRODUCT_TASK_BOARD_AND_UPDATE_TEMPLATE_zh.md`
  - 执行看板与任务回写模板。
- `CODEX_EXECUTION_PROMPTS_T1_T16_zh.md`
  - 面向 `Codex CLI` 的正式执行提示词。
- `PRODUCT_DECISION_LOG_zh.md`
  - 关键产品决策与取舍记录。
- `PRODUCT_VERSION_PACKAGING_AND_RELEASE_PLAN_zh.md`
  - 版本装箱与发布任务映射。
- `PRODUCT_TASK_DEPENDENCY_AND_RISK_REGISTER_zh.md`
  - 任务依赖、风险等级与启动条件。
- `PRODUCT_EVIDENCE_INDEX_AND_ACCEPTANCE_ASSETS_zh.md`
  - 验收证据与资产索引。
- `AI_ONBOARDING_QUICKSTART_zh.md`
  - AI 助手快速上手与阅读顺序。
- `CURRENT_CAPABILITIES_AND_BOUNDARIES_zh.md`
  - 当前主线能力面与边界定义。
- `CODE_ENTRYPOINTS_AND_OWNERSHIP_zh.md`
  - 代码入口与责任地图。

## 使用规则

1. 产品管理文档默认统一放在本目录。
2. 任务状态、版本、飞书留档更新时，优先更新本目录下对应源文档。
3. 新增 PM 文档时，必须同时补本 README 和主文档索引。
