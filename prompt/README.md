# Prompt Examples

This directory contains legacy public prompt examples. They are not runtime
instructions, owner-lane planning material, release evidence, or the source of
truth for current Tabrix browser automation.

Current browser automation behavior is defined by:

- `docs/TOOLS.md`
- `docs/CLI_AND_MCP.md`
- `skills/tabrix_browser/SKILL.md`

Some older prompts in this directory were originally written for historical
injection-helper workflows. Treat those files as archived examples unless they
are explicitly updated against the current public tool surface.
