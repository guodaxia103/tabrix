# Tabrix 能力层

## 目标

让 AI 助手清楚知道 Tabrix 能做什么，以及应该怎么最短路径完成任务。

## 典型能力

- 列出窗口与标签页
- 导航到目标页面
- 读取当前页面结构
- 提取正文内容
- 点击页面元素
- 填写表单
- 键鼠与滚动操作
- 页面截图
- 网络与控制台调试

## 最短成功路径

1. `get_windows_and_tabs`
2. `chrome_navigate`
3. `chrome_read_page`
4. `chrome_click_element` / `chrome_fill_or_select`
5. 结果验证

## 使用原则

- 先结构化读取，再操作
- 先精确点击，再考虑坐标
- 先当前真实 Chrome，再考虑其它浏览器环境
- 先单步确认，再组合复杂动作

## 不建议做的事

- 不要把 shell 作为默认浏览器控制路径
- 不要只靠截图盲点页面
- 不要在页面变化后继续复用旧 `ref`
